# 小米狗精灵设计文档

**日期：** 2026-05-30
**状态：** 演示设计
**范围：** 基于小米猫设计语言，创建小米狗精灵变体

---

## 1. 设计目标

基于小米猫精灵的设计语言，创建一个狗的变体形象。保持 MI 解构的设计理念，但用狗的特征重新诠释。

**核心差异：**
- 猫耳朝上 ╱╲（尖锐、优雅）→ 狗耳朝下 ╲ ╱（下垂、憨厚）
- 猫嘴 ω → 狗鼻 ▼
- 猫脸窄（灵动）→ 狗脸宽（亲和）

## 2. 视觉设计

### 2.1 核心概念：狗 = MI 解构（变体）

```
M = 狗耳 — 两个下垂弧形 ╲ ╱，比猫耳更圆润柔和
I = 狗尾 — 竖线 │，摇动频率更快（比猫更活泼）
超椭圆 = 狗脸 — ╭╮╰╯ 圆角括号，比猫脸略宽（更憨厚）
```

### 2.2 基础造型

```
  ╲        ╱        ← M 的下垂弧形 = 狗耳
   ╲──────╱         ← 耳朵连接处
   │  ●    ● │      ← 眼睛（● 实心圆）
   │    ▼    │      ← 狗鼻（▼ 三角形）
    ╲      ╱        ← 超椭圆脸下半部
     ╰──╯│          ← 超椭圆底部 + 身体
           │        ← I 的竖线 = 狗尾
```

### 2.3 尺寸约束

- **宽度：** 12字符（与小米猫一致）
- **高度：** 7行（与小米猫一致）

### 2.4 配色方案

与小米猫完全一致，固定小米橙。

| 元素 | 颜色 | 色值 |
|------|------|------|
| 主色调 | 小米橙 | `#FF6900` |
| 耳朵高光 | 浅橙 | `#FF8C33` |
| 眼睛 | 纯白 | `#FFFFFF` |
| 尾巴尖 | 深橙 | `#CC5500` |

## 5. 与小米猫的对比

| 特征 | 小米猫 | 小米狗 |
|------|--------|--------|
| 耳朵 | 朝上 ╱╲（尖锐） | 朝下 ╲ ╱（下垂） |
| 嘴巴 | ω（猫嘴） | ▼（狗鼻） |
| 脸型 | 窄（优雅） | 宽（憨厚） |
| 性格 | 灵动、神秘 | 憨厚、忠诚 |
| 尾巴 | 优雅摇摆 | 活泼摇摆 |
